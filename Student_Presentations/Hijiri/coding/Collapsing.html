What is a Collapsing Header?

A collapsing header is a style of website design that utilizes HTML, CSS, and possibly Java Script. The basic concept is that the header or banner is elevated on the z-axis. This allows for the other content to scroll below the specific banner. In our website, we will teach the user methods to create and customize their own collapsing header. Using resources such as W3Schools and DEVTips the user will be able to further their knowledge.

After you fully grasp the concept of the collapsing header, as a web designer, yo can create countless beautiful websites. Websites that not only look, but feel like it was done by a proffesional. The hardest part about making a collapsing header is the div tags. You must be certain that they are in order. If not the css IDs wont change properly
